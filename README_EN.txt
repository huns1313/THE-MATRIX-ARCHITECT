# THE MATRIX ARCHITECT v14.8

### High-Performance Item Filter for Project Diablo 2

This repository hosts the official configuration for **THE MATRIX ARCHITECT**, a precision-engineered item filter designed for maximum efficiency, visual clarity, and elite-tier farming.

---

## ## SYSTEM OPERATING MODES

The filter dynamically scales visibility based on your character level ($CLVL$) and current map environment:

* **[GOD] HYUUNNS:** High-end drops (HRs, Perfect Bases, Godly Uniques). Always visible with high-priority Sound IDs and map icons.
* **[HELLYA] HELL:** Engineered for Endgame farming. Strips away low-level clutter to highlight Elite bases and valuable Uniques.
* **[NM.] NIGHT:** Balanced visibility for Nightmare progression and early Hell transitions.
* **[N.] NORMA:** Raw ASCII mode for leveling and standard item visibility.

---

## ## CORE FEATURES

* **FIXED UNIQUE VISIBILITY:** Forced visibility for all Normal and Exceptional Uniques. Never miss a potential grail item again.
* **DYNAMIC GOLD FILTER:** Automatically hides small gold stacks as you level up to keep the battlefield clean.
* **"FUEL" MANAGEMENT (POTIONS):**
* **Rejuvenation Potions:** Labeled as **[Fuel]** in purple with dedicated sound alerts.
* **Health Potions:** Color-coded in **Tan** for instant visual recognition.


* **CRAFTING & BASES:** Advanced detection for Superior items, displaying $ED$ (Enhanced Damage) and $AR$ (Attack Rating) stats directly in the name.
* **SYSTEM EXCEPTION (RED):** Broken items or low-value socketed items are flagged in **RED** to prevent accidental pickups.

---

## ## INSTALLATION

Path: Navigate to C:\Program Files (x86)\Diablo II\ProjectD2\filters\local.

Deposit: Copy your .filter file into this local folder.

Play: Launch the game and select the filter in the settings.

Note: Make sure the file extension is .filter so the game engine detects it properly.
---

> **Note:** This filter is built for speed. It utilizes aggressive auto-hiding for non-essential items in high-tier maps to maximize clear speed and eliminate visual noise.
