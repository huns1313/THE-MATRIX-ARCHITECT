THE MATRIX ARCHITECT v14.8
Filtre d'objets haute performance pour Project Diablo 2
Ce dépôt contient la configuration officielle de THE MATRIX ARCHITECT, un filtre optimisé pour la clarté visuelle et la performance lors de vos sessions de farm intensives.

## MODES DE FONCTIONNEMENT
La visibilité des objets s'adapte dynamiquement à votre progression :

[GOD] HYUUNNS : Objets de très haute valeur. Alertes sonores prioritaires et marquage précis sur la carte.

[HELLYA] HELL : Configuration pour le contenu de fin de jeu (Endgame). Masque les objets obsolètes pour mettre en avant les bases Elite.

[NM.] NIGHT : Visibilité optimisée pour le mode Cauchemar.

[N.] NORMA : Affichage standard pour la phase de montée en niveau.

## POINTS FORTS :
CORRECTION DES UNIQUES : Visibilité forcée sur tous les objets Uniques (Normaux/Exceptionnels) pour garantir la sécurité de vos drops.

FILTRE D'OR DYNAMIQUE : Masquage automatique de l'or au sol en fonction du niveau du personnage.

SYSTÈME "FUEL" (POTIONS) :

Potions de Rajeunissement : Identifiées comme "Fuel" avec un code couleur violet.

Potions de Soin : Affichées en couleur Tan pour une distinction immédiate.

BASES ET ARTISANAT : Affichage détaillé des statistiques d'objets Supérieurs (ED / AR).

## INSTALLATION
Copiez le contenu du fichier matrix_architect.filter.

Remplacez le contenu de votre fichier default.filter (dossier PD2) par ce code.

Vérifiez l'activation du filtre dans les options du jeu.
