# THE MATRIX ARCHITECT v14.8

### Le filtre d'items le plus "sharp" pour Project Diablo 2

C'est ici que ça se passe pour la config la plus récente de **THE MATRIX ARCHITECT**. On a bâti ça pour que ça soit efficace, clair, pis que tu perdes pas ton temps avec de la scrap quand tu farm.

## ## LES MODES DU SYSTÈME

Le filtre s'ajuste tout seul selon ton niveau ($CLVL$) pis où tu te trouves :

* **[GOD] HYUUNNS :** Les gros drops (HRs, Bases Perfect, Uniques de fou). Toujours visibles avec le gros son pis l'icône sur la map.
* **[HELLYA] HELL :** Pour le farm sérieux. On cache la cochonnerie pour focuser sur les bases Elite pis les Uniques qui valent de quoi.
* **[NM.] NIGHT :** Le juste milieu pour le Nightmare pis le début du Hell.
* **[N.] NORMA :** Mode "ASCII" de base pour quand tu montes ton perso.

## ## POURQUOI C'EST L'BEST :

* **UNIQUES FIXÉS :** Les Uniques Normaux et Exceptionnels sont "forcés" visibles. On en manque plus aucun, big.
* **GESTION DU GOLD :** Plus ton perso est haut niveau, plus le filtre cache les petites piles de change pour pas polluer ton écran.
* **POTIONS (FUEL) :**
* **Full Rejuvs :** Marquées "Fuel" en mauve avec un son quand ça tombe.
* **Potions de vie :** En **Tan** (brun/beige) pour les spotter tout de suite.


* **BASES DE CRAFT :** Ça check direct les items Supérieurs avec les % de $ED$ et d' $AR$.
* **RED FLAGS :** Si c'est brisé ou inutile, ça sort en **ROUGE**.

## ## INSTALLATION

1. Copie le code de `matrix_architect.filter`.
2. Garroche ça dans ton fichier `default.filter` dans ton dossier de PD2.
3. Active ça in-game pis t'es prêt.

# THE MATRIX ARCHITECT v14.8

### Filtre d'objets haute performance pour Project Diablo 2

Ce dépôt contient la configuration officielle de **THE MATRIX ARCHITECT**, un filtre optimisé pour la clarté visuelle et la performance lors de vos sessions de farm intensives.

## ## MODES DE FONCTIONNEMENT

La visibilité des objets s'adapte dynamiquement à votre progression :

* **[GOD] HYUUNNS :** Objets de très haute valeur. Alertes sonores prioritaires et marquage précis sur la carte.
* **[HELLYA] HELL :** Configuration pour le contenu de fin de jeu (Endgame). Masque les objets obsolètes pour mettre en avant les bases Elite.
* **[NM.] NIGHT :** Visibilité optimisée pour le mode Cauchemar.
* **[N.] NORMA :** Affichage standard pour la phase de montée en niveau.

## ## POINTS FORTS :

* **CORRECTION DES UNIQUES :** Visibilité forcée sur tous les objets Uniques (Normaux/Exceptionnels) pour garantir la sécurité de vos drops.
* **FILTRE D'OR DYNAMIQUE :** Masquage automatique de l'or au sol en fonction du niveau du personnage.
* **SYSTÈME "FUEL" (POTIONS) :**
* **Potions de Rajeunissement :** Identifiées comme "Fuel" avec un code couleur violet.
* **Potions de Soin :** Affichées en couleur **Tan** pour une distinction immédiate.


* **BASES ET ARTISANAT :** Affichage détaillé des statistiques d'objets Supérieurs ($ED$ / $AR$).

## INSTALLATION (QUÉBEC)
Le Chemin : Ouvre ton dossier C:\Program Files (x86)\Diablo II\ProjectD2\filters\local.

Le Dépôt : Garroche ton fichier matrix_architect.filter direct dans ce dossier-là.

PLAY : Lance le jeu, va dans les options de filtres pis sélectionne-le.
